# Research Notes

このディレクトリには、ジユウノコンパスの診断ロジック、文言、UI、ミッション設計に使うDeep Research資料を保存しています。
今後、FIRE、取り崩し、生活防衛資金、固定費改善、働き方を緩める提案に関わる変更を行う場合は、該当資料を確認してから実装してください。

## 資料一覧

### [fire-withdrawal-japan.md](./fire-withdrawal-japan.md)

日本在住者向けのFIRE、取り崩し、4%ルール、NISA、iDeCo、年金、為替、外国課税、資産カバー率に関する調査です。

使いどころ:

- セミリタイア距離メーター
- 資産が生活費を何%支えるかの表示
- 2.5% / 3.0% / 3.5% / 4.0% の取り崩し率表示
- FIREや取り崩しに関する免責文
- NISA、iDeCo、年金、為替、外国課税に触れる補足文

注意:

- 4.0%は安全判定ではなく、米国過去データの参考線として扱う。
- FIRE達成を断定せず、生活費の一部を資産で支える考え方に翻訳する。

### [workstyle-options-japan.md](./workstyle-options-japan.md)

日本在住者が働き方を緩める現実的な選択肢を整理した調査です。
有給、残業整理、在宅勤務、時差出勤、業務量調整、異動、時短、週4勤務、転職準備、雇用形態変更、休職、相談窓口などを扱います。

使いどころ:

- 働き方を緩めるミッション
- 休息・相談優先タイプ
- 在職で負荷軽減タイプ
- 転職準備先行タイプ
- 仕事のしんどさが強いユーザーへの安全な文言
- 副業やフリーランス化を後回しにする判断

注意:

- 退職を最初の提案にしない。
- 疲弊が強い人には、副業より休息・制度確認・相談を優先する。
- 休職、傷病手当金、雇用保険、社内制度は個別差が大きいため断定しない。

### [emergency-fund-fixed-costs-japan.md](./emergency-fund-fixed-costs-japan.md)

日本在住者向けの生活防衛資金、貯金目安、固定費改善に関する調査です。
会社員3〜6か月、フリーランス6〜12か月、家族構成や固定費の重さによる調整、貯金ゼロからの段階目標、固定費改善の優先順位を扱います。

使いどころ:

- 生活防衛資金チェック
- 生活防衛資金のレンジ表示
- 1万円、3万円、10万円、0.5か月、1か月などの段階目標
- 固定費改善ミッション
- 住居費、保険、通信、サブスク、光熱費、ローン関連の優先順位

注意:

- 生活防衛資金は単一の正解金額として出さない。
- 不足を責めず、「いまは何段階目か」「次は何を目指すか」で表示する。
- 固定費は「今日できる小さな改善」と「大きく効く改善」に分ける。

## UIへの翻訳方針

研究資料はそのまま長文でUIに貼らず、以下の形に翻訳します。

- 短い説明文
- カード
- メーター
- 1〜3個のミッション
- 補足ツールチップ
- 結果タイプの理由
- 免責の短文

研究資料にない数値や制度を勝手に追加しないでください。
制度や税制など将来変わる可能性がある内容は、必要に応じて最新情報を確認し、「目安」「参考」「確認してください」と表現してください。
