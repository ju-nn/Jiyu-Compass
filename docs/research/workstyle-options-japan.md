# 日本在住者向け 働き方を緩める現実的な選択肢の調査報告

## 要点まとめ

公開中の「ジユウノコンパス」は、現在「働き方を軽くするための生活コンパス診断」として案内されており、今回の整理もその文脈に合わせて、「いきなり退職」ではなく、今の雇用を活かしながら自由度を上げる順番を中心にまとめています。 citeturn1view0

- いちばんリスクが低い緩め方は、**在職を維持したまま**、年休の確保、有給での回復、残業抑制、在宅勤務、時短・週四勤務、異動・職種変更の順で負荷を下げることです。こうした順番は、社会保険と職歴の連続性を保ちやすく、再就職でも説明しやすいです。 citeturn4search8turn19search0turn2search1turn23search4turn24search2turn25search8

- **時短勤務**は、法律上は育児・介護の場面で明確な制度がありますが、一般論としては「短時間正社員」や「多様な正社員」制度として会社が設計しているケースが現実的です。**週四勤務**は、厚生労働省の「選択的週休3日制」の整理でも、給与維持型と給与減少型の両方があり、会社ごとの制度設計差がかなり大きいです。 citeturn16search1turn16search0turn23search4turn24search2

- 派遣、契約社員、パート・アルバイトは、日本で十分に現実的な選択肢です。総務省統計局によると、2025年平均の非正規の職員・従業員は2128万人で、現職の雇用形態についた主な理由で最も多かったのは「自分の都合のよい時間に働きたいから」でした。つまり、「雇用形態をやや軽くする」は日本では珍しい逃げではなく、ごく普通に使われている調整手段です。 citeturn18search2turn18search5

- 反対に、**副業やフリーランス化は、疲弊が強い人の最初の一手にはしにくい**です。副業・兼業は原則認める方向が望ましいとされる一方、雇用される形の副業では労働時間の通算管理や健康管理が問題になりやすく、フリーランスは取引条件保護が強化されたとはいえ、収入変動、営業、請求、社会保険の自己管理が発生します。 citeturn27search0turn27search7turn3search3turn10search3

- しんどさが強い人には、**「辞めるか続けるか」より先に、「休む」「制度を確認する」「相談する」**を出す方が安全です。私傷病休職の定義や期間、復職の扱いは労基法に定めがなく就業規則次第で、健康保険の傷病手当金は一定条件で支給されます。相談先としては、厚労省の「こころの耳」や「労働条件相談ほっとライン」が使えます。 citeturn26search5turn4search3turn4search2turn4search1

### 比較の見方

以下の「収入」「安定性」「メンタル負荷」「再就職しやすさ」は、法令が点数化しているものではなく、**拘束時間、通勤、対人負荷、役割の曖昧さ、雇用の連続性**から見た一般的な傾向です。2024年4月以降、求人票では**業務内容の変更の範囲、就業場所の変更の範囲、有期契約の更新上限**などの明示が追加されているので、制度名だけでなく、求人の書き方まで見る前提で使うのが大事です。 citeturn14search4turn25search7turn14search3

## 辞める前に踏める現実的ステップ

### 会社員が辞める前に確認すべきチェックリスト

まず確認したいのは、**今の会社の中で負荷を下げる余地**です。有給は労働者が請求した時季に与えるのが原則で、「事業の正常な運営を妨げる」場合に時季変更ができるだけで、付与しないことはできません。残業についても、原則の上限は月45時間・年360時間で、特別条項があっても年720時間、複数月平均80時間以内などの制限があります。しんどさが強い人ほど、まず「違法・過剰・継続困難になっていないか」の棚卸しが先です。 citeturn4search8turn19search0turn19search6

- □ 年休の残日数と、直近3か月の残業時間を確認した  
- □ 休日対応、深夜連絡、通勤時間、会議量を含めた「消耗ポイント」を書き出した  
- □ 上司または人事に、**業務量の優先順位付け・締切調整・担当の入れ替え**を相談する準備をした  
- □ 在宅勤務、ハイブリッド勤務、時差出勤、時短、選択的週休3日、短時間正社員制度の有無を就業規則や社内制度で確認した  
- □ 異動公募、職務限定、勤務地限定、転勤有無、変更の範囲を確認した  
- □ 休職の条文、必要書類、復職条件、休職満了時の扱いを確認した citeturn2search1turn23search4turn24search6turn26search3turn26search5turn25search7

次に大事なのは、**退職前のお金と保険の確認**です。健康保険は、退職後に任意継続・国民健康保険・家族の被扶養者という三択があり、任意継続は退職前に被保険者期間が継続2か月以上、退職日の翌日から20日以内の申請が必要です。自己都合退職の基本手当は、2025年4月以降は待期7日後に原則1か月の給付制限があるため、辞めた瞬間にすぐ現金が入る前提にはできません。だからこそ、診断サイトでは「今の家計で何か月もつか」を先に見せる設計がかなり効きます。 citeturn15search0turn15search2turn22search1turn5search0

- □ 必須生活費を、家賃・食費・光熱費・通信費・保険・医療・最低返済額ベースで月額化した  
- □ 現預金とすぐ現金化できる資産で、何か月分の必須生活費を賄えるか計算した  
- □ 退職後の健康保険三択を比較した  
- □ 雇用保険の待期と給付制限を確認した  
- □ 離職票、住民税、社会保険料の切替タイミングを確認した citeturn15search2turn22search1turn22search0

最後に、**転職活動だけ先に始める**のはかなり合理的です。ジョブ・カードは在職者でも使え、無料のキャリアコンサルティングも受けられます。学び直しが必要な場合は、教育訓練給付金の対象講座を先に当たると、辞めてから慌てるよりずっと安全です。 citeturn13search0turn13search9turn13search1

- □ ジョブ・カードで職務棚卸しを始めた  
- □ ハローワークまたは求人サイトで、在宅勤務・転勤なし・職務限定の条件保存をした  
- □ 履歴書・職務経歴書を「今の会社にいるうちに」更新した  
- □ 1社だけでも応募またはカジュアル面談を入れてみた citeturn20search0turn31search0turn13search0turn13search9

### 固定費を下げて選択肢を増やす考え方

生活費の見直しは、退職ありきではなくても効きます。日本FP協会は、家計の現状把握、家計の収支確認表の活用、固定費の見直し、緊急時の備えの重要性を繰り返し示しています。特に固定費は、一度下げると効果が毎月継続するので、自由度を上げるうえでいちばんレバレッジが高い部分です。 citeturn30search5turn30search8turn30search4

診断サイトでは、「節約しろ」よりも、**“固定費を1つ下げると、週4勤務に近づく”**のように、自由度との交換として見せると受け入れられやすいです。たとえば、通信費、保険、家賃、サブスク、自動車維持費のどれを下げると、必要生活費が月いくら減り、何か月分の防衛資金が増えるかを可視化すると、退職を煽らずに選択肢を増やせます。これは、日本FP協会が示す「家計の見える化」や固定費見直しの考え方とも整合します。 citeturn30search5turn30search0turn30search8

## 低ストレス寄りの職種と求人条件

### 低ストレス寄りになりやすい仕事の見方

「低ストレス職」は職種名だけでは決まりません。実際には、**定型業務か、社内向けか、一次対応か、電話中心か、クレーム中心か、変更の範囲が広いか**でかなり変わります。厚労省の職業情報提供サイトを見ると、一般事務は定型的な事務処理、データ入力は入力と検査・確認、PCキッティングはマニュアルに沿ったセットアップ、ヘルプデスクは社内向けと社外向けで性質が大きく違い、コールセンターは顧客の声やクレームに直接さらされやすいことが分かります。 citeturn6search0turn6search2turn7search1turn17search1turn7search0

### 実装しやすいロジック例

```text
runway_months = (現預金 + すぐ現金化できる資産 × 0.7) / 必須生活費
life_impact = 睡眠・食事・出勤・家事・集中力の自己申告スコア
internal_options = 年休 / 在宅 / 時短 / 週四 / 異動 / 休職 の利用可能数

if life_impact が高い:
    state = "休息・相談優先"
elif runway_months < 3:
    state = "固定費見直し優先"
elif internal_options >= 2:
    state = "在職で負荷軽減交渉"
elif runway_months >= 3 and 転職準備度が中以上:
    state = "転職準備先行"
elif runway_months >= 6 or 資産収入が必須生活費の30%以上:
    state = "資産補完で就業量調整"
else:
    state = "固定費見直し優先"
```

このロジックのポイントは、**「辞めるべき／辞めるな」の判定をしないこと**です。代わりに、「今の会社で一段下げる」「お金を整える」「次の仕事を水面下で準備する」「回復を優先する」というミッションに変換します。そうすると、医療・労務・法律に踏み込みすぎず、しかも行動に落ちやすいです。就業規則、社会保険、雇用保険は個別事情で大きく変わるため、制度名だけ示して確認を促す設計が安全です。 citeturn26search5turn15search2turn22search1

### メンタル負荷が高い人への安全な表現
