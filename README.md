# ジユウノコンパス

働きすぎない暮らしに近づくために、お金・時間・逃げ道を静かに整理する生活コンパス診断です。

FIRE達成を競わせるサイトではなく、生活の余力を見て「次にできる小さな一歩」を決めるための小さな生活設計ツールです。

## プロダクトの核

- 主役は「30秒診断」と「今やる一歩」。
- 月の余力、生活防壁、急な出費への耐性、仕事のしんどさを見ます。
- FIRE目標額、年金、固定費シミュレーター、時間を取り戻す道具は補助として扱います。
- 収益導線は世界観の延長にあるものだけにします。

詳しい判断基準は [PRODUCT_PRINCIPLES.md](./PRODUCT_PRINCIPLES.md) に記録しています。
改善方針は [IMPROVEMENT_DECISIONS.md](./IMPROVEMENT_DECISIONS.md) にまとめています。

## 主な機能

### 30秒診断

年齢、貯金、月収（手取り）、月の生活費だけで、今の余力と最初の一歩を見ます。

### 詳細入力

投資しているお金、自分で払う保険料、年金が少なくなりそうな年数、奨学金、住宅ローン、車ローン、貯金・投資経験、お金の不安感、仕事のしんどさを追加できます。

### 今の状態

月の余力、貯蓄率、急な出費にそなえる貯金、仕事の負担、生活防壁の距離を表示します。

### 今やる一歩

赤字を止める、固定費を1つ見直す、先取り貯金を作る、仕事のしんどさを見える化するなど、状況に合わせた行動を1つ出します。

### 逃げ道を作る

固定費を下げたら、月の余力、年間で残るお金、3か月分の貯金までの距離がどう変わるかを試算できます。

### 時間を取り戻す道具

家事や小さな手間を軽くする道具・サービスを、毎月戻る時間と時間価値の目安で見ます。

## 開発

```bash
npm install
npm run dev
```

開発サーバーは通常 http://localhost:5173/ で起動します。

```bash
npm run test
npm run build
```

## 現在のアプリ境界

現行の本体は `src/App.tsx`、`src/components/compass/*`、`src/utils/compass.ts` を中心にしています。

旧ダッシュボードやゲーム寄りの機能は `archive/` に保管しています。再導入する場合は、必ず `PRODUCT_PRINCIPLES.md` に照らして、生活コンパスの核を壊さないものだけを選びます。
